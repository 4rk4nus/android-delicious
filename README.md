Delicious Android
=================

Delicious Android is an Android app which helps you access and save bookmarks via [Delicious](http://www.delicious.com/).

Features
--------

* Good looks
* Display recents
* Tag list
* Save bookmarks
* Automatically fetch page title
* Save url from clipboard
* Colored tag display (wow!)
* Sliding menu
